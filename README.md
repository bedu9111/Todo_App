# ✅ My To-Do List App

A simple and responsive **To-Do List Web Application** built using **pure HTML, CSS, and JavaScript**.  
This app helps users plan their day by managing daily tasks efficiently.

---

## 🚀 Features

✔️ Add new tasks  
✔️ Mark tasks as completed  
✔️ Delete individual tasks  
✔️ Display remaining tasks count  
✔️ Clear completed tasks  
✔️ Clean and user-friendly UI  
✔️ Responsive design  

---

## 🛠️ Tech Stack

- **HTML5** – Structure  
- **CSS3** – Styling  
- **JavaScript (Vanilla JS)** – Functionality  

No external libraries or frameworks are used.
